---
name: start
description: Start a new TokenGolf roguelike run — declare quest, class, effort, and budget
---

Run the TokenGolf start wizard to begin a new roguelike run:

```bash
tokengolf start
```

This opens an interactive wizard where you pick a quest, character class, effort level, and budget. If `tokengolf` is not installed globally, install it first: `npm install -g tokengolf && tokengolf install`.
