# TokenGolf — Claude Code Plugin

Gamify your Claude Code sessions. Track token efficiency, earn achievements, level up your prompting.

## What this plugin does

- **Auto-tracks every session** (Flow mode) — just `/exit` and see your scorecard
- **Live HUD** showing cost, budget progress, context load, and model class
- **60+ achievements** based on prompting patterns, tool usage, and efficiency
- **Emotion system** — mood feedback based on session state

## Commands

- `/tokengolf:scorecard` — View last run's scorecard
- `/tokengolf:stats` — Career stats dashboard
- `/tokengolf:config` — View or change settings

## Full TUI

For rich Ink-rendered views, install the npm package alongside:

```bash
npm install -g tokengolf
```
